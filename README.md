# Додаток для рецептів десертів
## Основні завдання при створенні додатку
* Спроектувати структуру бази даних;
  
* Розробити зручний інтерфейс з притриманням обраної кольорової гами;

* Реалізувати логіку пошуку та фільтрації рецептів;

* Створити алгоритм динамічного перерахунку інгредієнтів залежно від діаметра форми.

## Діаграма компонентів
<img width="2000" height="650" alt="Image" src="https://github.com/user-attachments/assets/e9a98134-b6a4-4341-8d5c-06a84f6a2a4e" />
На рисунку зображена діаграма компонентів, яка візуально показує як всі деталі програми взаємодіють між собою. 
Все починається з бази даних, там лежить інформація про рецепти і просто чекає запиту. 
Генератор контенту(код на Python), підключається до бази через SQL інтерфейс.
Після чого забирає потрібні рецепти і формує їх у файл data.js. 
Під час відкриття користувачем веб-інтерфейсу, браузер бачить файлове сховище, формулює запит і отримує чистий json, після чого сайт показує гарні картки з рецептами.
